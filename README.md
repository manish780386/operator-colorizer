# Operator Colorizer Pro 🎨

Colors all operators in every programming language!

## Features
- 🟠 Arithmetic operators (+, -, *, /, %) — Orange
- 🔵 Comparison operators (==, !=, >=, <=) — Cyan
- 🟢 Logical operators (&&, ||, !) — Green
- 🩷 Assignment operators (=, +=, -=) — Pink
- 🟣 Bitwise operators (&, |, ^, ~) — Purple
- 📊 Operator stats counter
- 🎨 Custom colors support
- ✅ Toggle ON/OFF from status bar

## Supported Languages
JavaScript, TypeScript, Python, Java, C++, C, C#, Go, Rust, PHP, Ruby, Swift, Kotlin, Dart, and more!

## Usage
1. Install the extension
2. Open Command Palette (Ctrl+Shift+P)
3. Select "Operator Colorizer Pro" theme
4. Enjoy colored operators!

## Commands
- `Operator Colorizer: Toggle ON/OFF` — Enable/disable
- `Operator Colorizer: Show Operator Stats` — Count operators in file
- `Operator Colorizer: Customize Colors` — Change operator colors
